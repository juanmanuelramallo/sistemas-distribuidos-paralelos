## Entregable
#### Versión optimizada
  * [Ver codigo fuente](entregable_optimizado.c)
  * Para ejecutar correr los siguientes comandos:
    * `chmod 777 entregable_optimizado.sh`
    * [`./entregable_optimizado.sh`](entregable_optimizado.sh)
    * El script se encarga de correr el algoritmo para los valores 512 y 1024 de tamaño de matriz
  * [Ver salida de consola](salida_entregable_optimizado.md) de la ejecución

#### Versión por bloques
  * [Ver codigo fuente](entregable_bloques.c)
  * Para ejecutar correr los siguientes comandos:
    * `chmod 777 entregable_bloques.sh`
    * [`./entregable_bloques.sh`](entregable_bloques.sh)
    * El script se encarga de correr el algoritmo para distintos valores de tamaño y cantidad de bloques para luego poder comparar la mejor opción para los tamaños 512 y 1024 de matriz
  * [Ver salida de consola](salida_entregable_bloques.md) de la ejecución
